# Amazon Beauty Product Recommendation System

## Project Overview

This project develops a personalized product recommendation system using the Amazon Beauty Reviews dataset. The objective is to improve customer experience by recommending products based on historical rating behavior and collaborative filtering techniques.

---

## Business Problem

Online marketplaces often struggle to provide relevant product recommendations, leading to lower customer engagement and missed sales opportunities. This project addresses that challenge by analyzing customer rating behavior to generate personalized recommendations.

---

## Dataset

- Amazon Beauty Reviews Dataset
- Over 1 million customer ratings
- Features include:
  - User ID
  - Product ID
  - Rating
  - Timestamp
